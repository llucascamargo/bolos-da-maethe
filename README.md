Bolos da Maethe

Sobre o projeto

Projeto focado em front-end, com layout moderno e visual atrativo para uma confeitaria.

Desenvolvedores

Projeto desenvolvido em colaboração por:

- Lucas Camargo (@llucascamargo)
- Heitor Devequi (@dvkheitor)

Ambos participaram conjuntamente do desenvolvimento, design e estrutura do projeto.

Inspiração

O design deste projeto foi inspirado no site da Sodiê Doces, com adaptações para um estilo próprio.

O nome "Bolos da Maethe" foi criado como uma referência criativa inspirada em "Bolos da Madre", sendo "Maethe" uma alusão ao nome da namorada do criador de conteúdo Alanzoka.

Este projeto tem fins educacionais e de portfólio.
